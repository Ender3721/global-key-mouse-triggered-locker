# 键鼠全局触发式锁屏器
为了保持电脑常开（方便下载什么的），又避免被他人使用而制作；本人很不会编程，大部分代码都是gpt写的，创建这个项目只是为了研究github，项目软件本身可以做到：①直接使用，点击后监视鼠标和键盘 ②输入一个时间，输入后立即：在这个时间内程序保存运行，如果键鼠有任何活动，则锁屏 ③时间结束，即锁屏。其他：以确保win10可运行

## 运行说明

只需要 `zwitmoniter4.0.exe` 文件即可运行。
### 安装和使用

1. 下载最新版本的 `.exe` 文件。
2. 双击运行安装程序，按照提示完成安装。
